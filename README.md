#TimerClock

TiemrCLock is a simple to use Countdown / Stopwatch / Clock app

![TimerClock Screenshot](http://i.imgur.com/TX9XiHg.png)

###Features

* Countdown to time
* Countup from time
* Countdown timer
* Stopwatch
* Clock
